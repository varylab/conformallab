Nonlinear solver (SNES) options -------------------------------------------------
  -sin_fun_snes_type Nonlinear solver method:(one of) ls tr test (SNESSetType)
  -sin_fun_snes_view: Print detailed information on solver used (SNESView)
  -sin_fun_snes_stol <1e-008>: Stop if step length less then (SNESSetTolerances)
  -sin_fun_snes_atol <1e-050>: Stop if function norm less then (SNESSetTolerances)
  -sin_fun_snes_rtol <1e-008>: Stop if decrease in function norm less then (SNESSetTolerances)
  -sin_fun_snes_max_it <50>: Maximum iterations (SNESSetTolerances)
  -sin_fun_snes_max_funcs <10000>: Maximum function evaluations (SNESSetTolerances)
  -sin_fun_snes_max_fail <1>: Maximum failures (SNESSetTolerances)
  -sin_fun_snes_converged_reason: Print reason for converged or diverged (SNESSolve)
  -sin_fun_snes_ksp_ew: <FALSE> Use Eisentat-Walker linear system convergence test (SNESKSPSetUseEW)
  -sin_fun_snes_ksp_ew_version <2>: Version 1, 2 or 3 (SNESKSPSetParametersEW)
  -sin_fun_snes_ksp_ew_rtol0 <0.3>: 0 <= rtol0 < 1 (SNESKSPSetParametersEW)
  -sin_fun_snes_ksp_ew_rtolmax <0.9>: 0 <= rtolmax < 1 (SNESKSPSetParametersEW)
  -sin_fun_snes_ksp_ew_gamma <1>: 0 <= gamma <= 1 (SNESKSPSetParametersEW)
  -sin_fun_snes_ksp_ew_alpha <1.61803>: 1 < alpha <= 2 (SNESKSPSetParametersEW)
  -sin_fun_snes_ksp_ew_alpha2 <1.61803>: alpha2 (SNESKSPSetParametersEW)
  -sin_fun_snes_ksp_ew_threshold <0.1>: 0 < threshold < 1 (SNESKSPSetParametersEW)
  -sin_fun_snes_no_convergence_test: Don't test for convergence (None)
  -sin_fun_snes_monitor_cancel: Remove all monitors (SNESMonitorCancel)
  -sin_fun_snes_monitor <stdout>: Monitor norm of function (SNESMonitorSet)
  -sin_fun_snes_ratiomonitor <stdout>: Monitor ratios of norms of function (SNESMonitorSetRatio)
  -sin_fun_snes_monitor_short <stdout>: Monitor norm of function (fewer digits) (SNESMonitorSet)
  -sin_fun_snes_monitor_solution: Plot solution at each iteration (SNESMonitorSolution)
  -sin_fun_snes_monitor_solution_update: Plot correction at each iteration (SNESMonitorSolutionUpdate)
  -sin_fun_snes_monitor_residual: Plot residual at each iteration (SNESMonitorResidual)
  -sin_fun_snes_monitor_draw: Plot function norm at each iteration (SNESMonitorLG)
  -sin_fun_snes_fd: Use finite differences (slow) to compute Jacobian (SNESDefaultComputeJacobian)
  SNES Line search options
  -sin_fun_snes_ls_alpha <0.0001>: Function norm must decrease by (None)
  -sin_fun_snes_ls_maxstep <1e+008>: Step must be less than (None)
  -sin_fun_snes_ls_steptol <1e-012>: Step must be greater than (None)
  -sin_fun_snes_ls <cubic> (choose one of) basic basicnonorms quadratic cubic
Preconditioner (PC) Options -------------------------------------------------
  -sin_fun_pc_type Preconditioner:(one of) none jacobi pbjacobi bjacobi sor lu shell mg
      eisenstat ilu icc cholesky asm ksp composite redundant nn mat fieldsplit galerkin openmp tfs (PCSetType)
  ILU Options
  -sin_fun_pc_factor_levels <0>: levels of fill (PCFactorSetLevels)
  -sin_fun_pc_factor_in_place: do factorization in place (PCFactorSetUseInPlace)
  -sin_fun_pc_factor_diagonal_fill: Allow fill into empty diagonal entry (PCFactorSetAllowDiagonalFill)
  -sin_fun_pc_factor_reuse_fill: Reuse fill ratio from previous factorization (PCFactorSetReuseFill)
  -sin_fun_pc_factor_reuse_ordering: Reuse previous reordering (PCFactorSetReuseOrdering)
  -sin_fun_pc_factor_shift_nonzero: Shift added to diagonal (PCFactorSetShiftNonzero)
  -sin_fun_pc_factor_shift_nonzero <1e-012>: Shift added to diagonal (PCFactorSetShiftNonzero)
  -sin_fun_pc_factor_shift_positive_definite: Manteuffel shift applied to diagonal (PCFactorSetShiftPd)
  -sin_fun_pc_factor_zeropivot <1e-012>: Pivot is considered zero if less than (PCFactorSetZeroPivot)
  -sin_fun_pc_factor_use_drop_tolerance <-2>: <dt,dtcol,maxrowcount> (PCFactorSetUseDropTolerance)
  -sin_fun_pc_factor_fill <1>: Expected fill in factorization (PCFactorSetFill)
  -sin_fun_pc_factor_nonzeros_along_diagonal: Reorder to remove zeros from diagonal (PCFactorReorderForNonzeroDiagonal)
  -sin_fun_pc_factor_mat_ordering_type Reorder to reduce nonzeros in ILU:(one of) natural nd 1wd rcm qmd rowlength (PCFactorSetMatOrderingType)
  -sin_fun_pc_factor_pivot_in_blocks: <TRUE> Pivot inside matrix blocks for BAIJ and SBAIJ (PCFactorSetPivotInBlocks)
Krylov Method (KSP) Options -------------------------------------------------
  -sin_fun_ksp_type Krylov method:(one of) cg cgne stcg gltr richardson chebychev gmres tcqmr
      bcgs bcgsl cgs tfqmr cr lsqr preonly qcg bicg fgmres minres symmlq lgmres lcd (KSPSetType)
  -sin_fun_ksp_max_it <10000>: Maximum number of iterations (KSPSetTolerances)
  -sin_fun_ksp_rtol <1e-005>: Relative decrease in residual norm (KSPSetTolerances)
  -sin_fun_ksp_atol <1e-050>: Absolute value of residual norm (KSPSetTolerances)
  -sin_fun_ksp_divtol <10000>: Residual norm increase cause divergence (KSPSetTolerances)
  -sin_fun_ksp_converged_use_initial_residual_norm: Use initial residual residual norm for computing relative convergence (KSPDefaultConvergedSetUIRNorm)
  -sin_fun_ksp_converged_use_min_initial_residual_norm: Use minimum of initial residual norm and b for computing relative convergence (KSPDefaultConvergedSetUMIRNorm)
  -sin_fun_ksp_knoll: <FALSE> Use preconditioner applied to b for initial guess (KSPSetInitialGuessKnoll)
  -sin_fun_ksp_norm_type <preconditioned> (choose one of) NO PRECONDITIONED UNPRECONDITIONED NATURAL
  -sin_fun_ksp_diagonal_scale: Diagonal scale matrix before building preconditioner (KSPSetDiagonalScale)
  -sin_fun_ksp_diagonal_scale_fix: Fix diagonaled scaled matrix after solve (KSPSetDiagonalScaleFix)
  -sin_fun_ksp_constant_null_space: Add constant null space to Krylov solver (KSPSetNullSpace)
  -sin_fun_ksp_converged_reason: Print reason for converged or diverged (KSPSolve)
  -sin_fun_ksp_monitor_cancel: Remove any hardwired monitor routines (KSPMonitorCancel)
  -sin_fun_ksp_monitor <stdout>: Monitor preconditioned residual norm (KSPMonitorSet)
  -sin_fun_ksp_monitor_solution: Monitor solution graphically (KSPMonitorSet)
  -sin_fun_ksp_monitor_true_residual <stdout>: Monitor preconditioned residual norm (KSPMonitorSet)
  -sin_fun_ksp_monitor_singular_value <stdout>: Monitor singular values (KSPMonitorSet)
  -sin_fun_ksp_monitor_short <stdout>: Monitor preconditioned residual norm with fewer digits (KSPMonitorSet)
  -sin_fun_ksp_monitor_draw: Monitor graphically preconditioned residual norm (KSPMonitorSet)
  -sin_fun_ksp_monitor_draw_true_residual: Monitor graphically true residual norm (KSPMonitorSet)
  Pick at most one of -------------
    -sin_fun_ksp_left_pc: Use left preconditioning (KSPSetPreconditionerSide)
    -sin_fun_ksp_right_pc: Use right preconditioning (KSPSetPreconditionerSide)
    -sin_fun_ksp_symmetric_pc: Use symmetric (factorized) preconditioning (KSPSetPreconditionerSide)
  -sin_fun_ksp_compute_singularvalues: Compute singular values of preconditioned operator (KSPSetComputeSingularValues)
  -sin_fun_ksp_compute_eigenvalues: Compute eigenvalues of preconditioned operator (KSPSetComputeSingularValues)
  -sin_fun_ksp_plot_eigenvalues: Scatter plot extreme eigenvalues (KSPSetComputeSingularValues)
  KSP GMRES Options
  -sin_fun_ksp_gmres_restart <30>: Number of Krylov search directions (KSPGMRESSetRestart)
  -sin_fun_ksp_gmres_haptol <1e-030>: Tolerance for exact convergence (happy ending) (KSPGMRESSetHapTol)
  -sin_fun_ksp_gmres_preallocate: Preallocate Krylov vectors (KSPGMRESSetPreAllocateVectors)
  Pick at most one of -------------
    -sin_fun_ksp_gmres_classicalgramschmidt: Classical (unmodified) Gram-Schmidt (fast) (KSPGMRESSetOrthogonalization)
    -sin_fun_ksp_gmres_modifiedgramschmidt: Modified Gram-Schmidt (slow,more stable) (KSPGMRESSetOrthogonalization)
  -sin_fun_ksp_gmres_cgs_refinement_type <REFINE_NEVER> (choose one of) REFINE_NEVER REFINE_IFNEEDED REFINE_ALWAYS
  -sin_fun_ksp_gmres_krylov_monitor: Plot the Krylov directions (KSPMonitorSet)
  -sin_fun_ksp_view: View linear solver parameters (KSPView)
